# WildBliss Tours

A modern safari tour booking platform built with Laravel and React, showcasing Kenya's incredible wildlife destinations.

## Tech Stack

- **Backend**: Laravel 12
- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **Full-Stack**: Inertia.js
- **Authentication**: Laravel Fortify
- **Icons**: Lucide React

## Features

- Responsive landing page with hero video
- Tour browsing and destination exploration
- Contact form with validation
- Centralized brand color system
- Mobile-friendly navigation
